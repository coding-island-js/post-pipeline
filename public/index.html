Post Pipeline

One shared view of where every episode is in post-production — the picture master, the Atmos mix, captions, subtitles, VFX and more — from first edit all the way to delivery on the platform.

Think of it as the producer's source of truth that sits on top of Avid, Frame.io, Resolve and Pro Tools.

How work moves through post-production

    The board

    Each column is a stage of post. A card is one deliverable for an episode. Drag a card to move it to a new stage, change its status inline, or click ✎ to edit everything.

    Where this fits in a real post workflow

    Post teams already live in specialist tools — this doesn't replace them, it tracks across them so a producer can answer "where is everything?" in one glance.

    • Avid Media Composer — editorial / picture lock
    • ShotGrid + Nuke — VFX shots
    • DaVinci Resolve — color / DI
    • Pro Tools — mix, ADR, M&E stems
    • Frame.io — review & approval notes
    • Aspera / Signiant — delivery transfers (IMF)

    Under the hood

    The board is a web page on Netlify. The data lives in an Airtable base. When you change a status here, a small serverless function writes it straight to Airtable — the very same record a coordinator sees in the Airtable grid. One source of truth, two ways to look at it.
